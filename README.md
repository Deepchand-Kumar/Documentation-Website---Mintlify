# Documentation-Website---Mintlify
This project is a desktop-first recreation of the official Mintlify documentation website, developed using only HTML and CSS.

The objective of this assignment is to focus on content structure, readability, layout accuracy, and visual consistency with the original website.

Original Website: https://www.mintlify.com/
Assignment Overview
This project was created following the given assignment guidelines:

✅ Desktop-first layout
✅ HTML & CSS only
✅ No JavaScript
✅ No Tailwind / Frameworks
✅ No Animations
✅ No AI-generated code
✅ Static documentation-style layout
✅ Focus on readability and structure
